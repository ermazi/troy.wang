    名词解释

    Meta-Arguments

    cli可以在resource中使用meta-arguments，来改变资源的行为：

    • depends_on 显式指定依赖
    • count 根据count指定多个资源，可以使用count.index引用当前count
    • for_each 根据map或者set来创建多个资源，key用each.key，value用each.value
    • provider 选择一个非默认的provider
    • lifecycle 定义生命周期
    • provisioner and connection 资源创建成功以后做一些事，比如注册啊，执行配置管理任务啊之类的
